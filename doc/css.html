CSS 내용 정리

DISPLAY

요소가 화면에서 어떻게 표시되는지를 지정하는 속성이다.

  • none

    요소를 화면에서 보이지 않도록 숨긴다. (transition 사용 불가)

  • block

    요소를 블록 레벨 요소로 표시. 즉, 요소가 한 줄을 차지하고, 줄 바꿈이 발생하며, 너비와 높이를 설정할 수 있다.

  • inline

    요소를 인라인 요소로 표시, 요소가 텍스트와 같이 흐르며, 너비와 높이를 설정할 수 없다.

  • inline-block

    요소를 인라인 레벨 요소처럼 흐르면서, 블록 레벨 요소처럼 너비와 높이를 설정할 수 있는 형태로 표시 한다.

  • table

    요소를 테이블 레이아웃으로 표시
    table-caption : <caption> 요소처럼 표현
    table-column-group : <colgroup> 요소처럼 표현
    table-header-group : <thead> 요소처럼 표현
    table-footer-group : <tfoot> 요소처럼 표현
    table-row-group : <tbody> 요소처럼 표현
    table-cell : <td> 요소처럼 표현
    table-column : <col> 요소처럼 표현
    table-row : <tr> 요소처럼 표현

  • flex

    요소를 플렉스 박스 레이아웃으로 표시
    flex-direction : Flex Items의 주 축을 설정 ("row", "column", "reverse" 등)
    flex-wrap : Flex Items의 여러 줄 바꿈 설정 ("nowrap", "wrap", "reverse" 등)
    justify-content : Flex Items의 주 축을 정렬
    ("flex-start", "flex-end", "center", "space-between", "space-around" 등)
    align-content : 축의 정렬 방법을 설정
    ("justify-content와 동일")
    align-items : 축에서 Items의 정렬 방법
    ("stretch", flex-start", "flex-end", "center", "baseline" 등)

    예시코드

    .flex {
           display: flex;
           gap: 30px;
           justify-content: space-between;
    }

  • grid

    요소를 그리드 레이아웃으로 표시
    grid-row : grid-row-xxx의 단축 속성(행 시작/끝 위치)
    ("grid-row-start", "grid-row-end")
    grid-column : grid-column-xxx의 단축 속성(열 시작/끝 위치)
    ("grid-column-start", "grid-column-end")
    grid-area : 영역(Area) 이름을 설정하거나, grid-row와 grid-column의 단축 속성

    예시코드

    .grid {
           display: grid;
           gap: 30px;
           grid-template-columns: 1fr 1fr 1fr;
    }

  • float

    복잡한 형태의 레이아웃을 구성하는데 필요한 핵심 속성으로 특정 요소를 떠있게, 흐르도록, 부유하게 하도록 하는 속성
    "left", "right" 속성으로 제어가 가능하고(부모에 가상요소를 이용해 "clear:both"사용해야한다.)

    예시코드

    .float부모 .float::after {
           content: "";
           display: block;
           clear: both;
    }

    .float {
           float: left;
           width: 50%;
    }

POSITION

요소의 위치를 지정하는 속성으로, 화면 상에서 어떻게 배치될지를 조절할 수 있다.

  • static

    요소를 일반적인 문서 흐름에 따라 배치합니다.

  • relative

    상대적인 위치로 이동할 수 있으며, 기준점이 된다.

  • absolute

    요소를 문서 흐름에서 제거하고, 부모 요소나 가장 가까운 위치 지정 조상 요소를 기준으로 절대적인 위치로 배치
    요소 중앙 배치 { transform: translate(-50%,-50%); }

  • fixed

    요소를 뷰포트를 기준으로 고정 위치로 배치(항상 고정된 값, 스크롤 영향 X)

  • sticky

    스크롤이 특정 임계값을 넘어가면 고정 위치로 배치(스크롤 영향을 받아 동적으로 위치 변동)

  • z-index

    "absolute", "fixed", "sticky" 상태에서 쓸수 있으며 숫자가 높은순으로 위로 배치된다.

    • 예시코드

    .position {
           position: fixed;
           top: 0;
           left: 0;
           width: 100%;
           height: 100px;
           z-index: 999;
    }

TEXT

텍스트의 스타일과 서체에 관련된 속성들을 설정하는 데 사용한다.

  • text-align

    텍스트의 가로 정렬을 설정
    ("left", "center", "right", "justify")

  • text-decoration

    텍스트의 장식을 설정
    ("none", "underline", "overline", "through ")

  • text-transform

    텍스트의 대소문자 변환을 설정
    ("none", "uppercase", "lowercase", "capitalize ")

  • text-shadow

    텍스트의 그림자를 설정
    ("가로" "세로" "번짐" "색상")

  • text-overflow

    텍스트의 말줄임표 설정
    ("clip", "visible", "ellipsis")

  • font-size

    텍스트의 글꼴 크기를 설정

  • font-weight

    텍스트의 글꼴 굵기를 설정

  • font-family

    텍스트의 글꼴 설정

  • font-style

    텍스트의 글꼴 스타일 설정

  • line-height

    텍스트의 줄 높이를 설정

  • letter-spacing

    텍스트의 글자 간격을 설정

  • letter-spacing

    텍스트의 글자 간격을 설정

  • color

    텍스트의 글자 색을 설정

  • word-break

    텍스트의 줄바꿈을 설정

    ("break-all", "keep-all", "break-word")
    • 예시코드

    .text {
           text-align: center;
           text-decoration: none;
           text-transform: none;
           text-shadow : 0 0 3px #000;
           text-overflow : ellipsis;

           font-size: 16px;
           font-weight: 300;
           font-family: 'Noto Sans KR';
           font-style: normal;

           line-height: 1;
           letter-spacing: 0.2px;
           color: #fcfcfc;
           word-break: keep-all;
    }

SPACE

요소의 여백 margin, padding, box-sizing

  • margin

    요소의 외부 여백을 설정하는 속성
    요소와 주변 요소 간의 공간을 조절하는 속성
    요소의 외부 여백을 통해 요소의 위치와 주변 요소와의 간격을 조절하는 속성
    ("margin: 0 auto; 속성은 가운데 정렬됨")

  • padding

    요소의 내부 여백을 설정하는 속성
    요소 내부의 내용과 테두리 사이의 공간을 조절하는 속성
    요소의 내부 여백을 통해 요소의 내용과 테두리 사이의 간격을 조절하는 속성

  • box-sizing: border-box

    너비(width)와 높이(height) 속성에 값을 지정할 때 콘텐츠 영역만을 고려하게 되어, 테두리와 패딩의 크기를 따로 고려할 필요가 없다.
    패딩과 테두리의 크기가 요소의 크기에 영향을 미치지 않기 때문에, 요소 간의 간격이 일정하게 유지된다

    • 예시코드

    .inner {
           margin: 0 auto;
           padding: 50px 0;
    }

LINE

박스의 선 이나, 테두리 등

  • border

    테두리의 선을 설정하는 속성
    ("solid", "dotted", "dashed", "double", "groove", "ridge", "inset", "outset")

  • border-radius

    테두리의 선을 둥글게만드는 속성
    ("px", "%")

  • outline

    요소 콘텐츠의 밖에 그려지며 절대 공간을 차지하지 않는다.

    • 예시코드

    .button {
           border: 1px solid #ddd;
           border-radius: 50%;
           outline: none;
    }

BACKGROUND

요소의 배경을 스타일링하는 데 사용되는 속성

  • background-color

    배경의 색상을 지정

  • background-image

    배경에 사용할 이미지 지정
    ("background-image: url(../img.jpg)")

  • background-repeat

    배경 이미지의 반복 여부를 설정
    ("repeat", "no-repeat")

  • background-position

    배경 이미지의 위치를 조정
    ("X축", "Y축")

  • background-size

    배경 이미지의 크기를 조정
    ("contain", "cover")

  • gradient

    배경 그라데이션 효과를 지정
    ("background: linear-gradient", "background: radial-gradient")

    • 예시코드

    .background-img {
           background-image: url(../img.jpg);
           background-repeat: no-repeat;
           background-position: center center;
           background-size: cover;
    }

    예시코드 (통합)

    .background {
           background: url(../img.jpg) no-repeat center center / cover;
    }

    예시코드 (gradient)

    .background {
           background: linear-gradient(to bottom, #ff0000 10%, #00ff00);
    }
