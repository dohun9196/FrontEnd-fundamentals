HTML 내용 정리

HEAD

기계가 식별할 수 있는 문서 정보(메타데이터)를 담고있다.

  • meta

    검색엔진 키워드, 웹페이지 설명, 뷰표트 등

  • title

    브라우저의 제목 표시줄이나 페이지 탭에 보이는 문서 제목을 정의

  • link

    현재 문서와 외부 리소르의 관계를 명시(stylesheet, favicon 등)

  • script

    데이터와 실행 가능한 코드를 문서에 포함(JS, JSON 등)

예시코드

<!DOCTYPE html >
<html lang="ko" >
<head >
<meta charset="UTF-8" >
<meta http-equiv="X-UA-Compatible" content="IE=edge">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<title>documentl</title >
<link rel="stylesheet" href="./css.css" >
<script src="./js.js" defer><script>
</head >

BODY

body 내에 들어가는 여러가지 태그

  • heading

    h1~h6 태그까지 존재한다.
    h1으로 시작해서 순차적으로 작성해야하며, 페이지당 h1은 1개만 존재해야한다.

    예시코드

    <h1>제목1</h1>
    <h2>제목2</h2>
    <h3>제목3</h3>
    <h4>제목4</h4>
    <h5>제목5</h5>
    <h6>제목6</h6>

  • text

    p : paragraph 태그로, 단락을 표시
    span : 인라인 요소를 감싸 스타일링 하거나, 텍스트의 의미적 구분을 담당
    strong, b : 텍스트를 굵게 표시
    em : 텍스트를 italic으로 표시
    i :텍스트를 italic으로 표시 하거나 아이콘 라이브러리를 사용
    br : break 태그로, 줄바꿈을 표시
    u : underline 태그로, 텍스트에 밑줄을 표시
    s : strikethrough 태그로, 텍스트에 취소선을 표시
    sup : superscript 태그로, 텍스트를 위첨자로 표시
    sub : subscript 태그로, 텍스트를 아래첨자로 표시
    a : anchor 태그로, 링크를 생성하며, 다른 웹페이지나 파일로 연결한다.

  • list

    ul : unordered list(순서없는) 태그로, 점 또는 기호로 항목을 나열
    ol : ordered list(순서있는) 태그로, 숫자나 알파벳 등으로 항목을 나열
    li : list item(목록의 항목) 태그로, "ul", "ol" 태그내에서 사용.

    예시코드

    <ul>
       <li>리스트01</li>
       <li>리스트02</li>
       <li>리스트03</li>
    </ul>
    <ol>
       <li>리스트01</li>
       <li>리스트02</li>
       <li>리스트03</li>
    </ol>

  • img

    img : image 태그로, 이미지 파일의 경로를 지정하여 웹 페이지에 이미지를 삽입

  • table

    thead : 테이블 헤더
    tbody : 테이블 본문
    tfoot : 테이블 푸터
    tr : 테이블 행
    th : 테이블 헤더 셀
    td : 테이블 데이터 셀

    예시코드

    <table>
       <thead>
            <tr>
                <th scope="col" >Lorem1.</th>
                <th scope="col" >Lorem2.</th>
            </tr>
        </thead>
       <tbody>
            <tr>
                <th scope="row" >Lorem1.</th>
                <td>text.</td>
            </tr>
            <tr>
                <th scope="row" >Lorem1.</th>
                <td>text.</td>
            </tr>
        </tbody>
    </table>

  • form

    form : action" 속성을 통해 데이터를 전송할 서버의 URL을 지정하고, "method" 속성을 통해 데이터 전송 방식을 지정
    input : 사용자로부터 텍스트, 숫자, 이메일 등의 값을 입력받는 입력 폼 요소
    type : 속성을 사용하여 입력의 종류를 지정
    select : 하나의 값을 선택할 수 있는 드롭다운 목록 형태의 입력 폼 요소
    textarea : 여러 줄의 텍스트를 입력받는 입력 폼 요소
    button : 클릭 가능한 버튼을 생성하는 입력 폼 요소
    checkbox : 체크 박스 버튼을 생성하는 입력 폼 요소
    radio : 라디오 버튼을 생성하는 입력 폼 요소로

    예시코드

    <form action="/submit" method="POST" >
       <input type="text" name="username" >
       <input type="password" name="password" >
       <input type="submit" value="제출" >
    </form >

semantic

웹 페이지의 구조와 의미를 명확하게 표현하기 위해 HTML5에서 도입된 태그

  • header

    웹 페이지나 섹션의 머리말

  • nav

    웹 페이지의 네비게이션 링크

  • aside

    웹 페이지의 별도의 내용 영역

  • main

    웹 페이지의 주요 내용

  • article

    독립적인 내용

  • section

    웹 페이지의 구획(section)

  • figure

    이미지, 도표, 삽화 등의 그래픽 컨텐츠

  • footer

    웹 페이지나 섹션의 바닥글

  • address

    웹 페이지의 주소나 연락처

예시코드

<header>
   <h1> 제목or로고 </h1>
   <nav>
       <ul>
           <li> 링크 </li>
       </ul>
   </nav>
</header>
<main>
   <section>
       <div>
           <figure>
               <img src="./img.jpg" alt="이미지">
           </figure>
       </div>
   </section>
</main>
<footer>
   <address>
       <span> 주소 <span>
       <span> 연락처 <span>
   </address>
</footer>
